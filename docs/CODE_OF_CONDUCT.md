# 行为准则

## 我们的承诺

为了营造一个开放友好的社区环境，作为 Tarn 社区的参与者和维护者，我们承诺：

- 🟢 **友善包容**：以礼待人，尊重不同背景和经验
- 🟢 **建设性**：以解决问题为导向，提供有价值的反馈
- 🟢 **耐心**：维护者是个人开发者，请给予合理响应时间
- 🟢 **尊重隐私**：不在公开场合泄露他人或自己的敏感信息
- 🟢 **聚焦主题**：讨论围绕 Tarn 本身，不引入无关话题

## 不可接受的行为

- 🔴 人身攻击、骚扰、侮辱性言论
- 🔴 针对性别、性取向、种族、宗教、国籍的歧视
- 🔴 公开泄露他人隐私信息
- 🔴 恶意刷屏、广告、垃圾信息
- 🔴 政治话题、引战讨论
- 🔴 **要求公开源代码、质疑闭源决策**（Tarn 是闭源项目，这是作者的有意选择）
- 🔴 **在公开 Issue 报告安全漏洞**（请按 [SECURITY.md](./docs/SECURITY.md) 流程私密报告）
- 🔴 威胁、恐吓行为

## 维护者责任

维护者有权：

- 关闭违反本准则的 Issue / 评论
- 删除违规内容
- 对违规者进行警告或 block
- 拒绝处理不友善的请求

维护者承诺：

- 公正处理每一条 Issue
- 不因个人喜好关闭合理的 Bug 报告
- 及时回复（在能力范围内）

## 适用范围

本准则适用于 Tarn 的所有社区空间，包括：

- GitHub Issues
- GitHub Discussions（如有）
- 官方公布的社交群组（如有）

## 举报

如发现违反本准则的行为，请通过以下方式举报：

- 在相关 Issue 下 @ 维护者
- 在仓库新建 Issue（标题前缀 `[COC]`）描述事件

请勿在公开场合与违规者对骂，交给维护者处理。

## 致谢

本行为准则参考了 [Contributor Covenant](https://www.contributor-covenant.org/) 的精神，并根据 Tarn 项目的实际情况（闭源、个人维护）做了简化。
